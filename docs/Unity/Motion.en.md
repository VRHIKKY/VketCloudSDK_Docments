
# Motion
![Motion](img/MotionJP.jpg)

Motion lets you play the motion set for the your avatar inside HEOWorldSetting. Specify the motion's name in the Motion Name.

|  Label |  function  |
| ----   | ---- |
| Motion Name | Set the motion's name you wish to play. |
