
# HEOAudio
![HEOAudio](img/HEOAudio.jpg)


|  Label |  function  |
| ----   | ---- |
| Audio Type | BGM、SE、SystemSEから選択できます。 |
| Audio Clip | オーディオファイルを指定します。 |
| loop | 繰り返し再生するかどうか設定します。 |
| autoplay | ワールドBGMのように、入室後から自動で再生する場合に使用します。 |

なお、BGMは以下の点に注意が必要です。

- 動画再生時には、動画の音声が優先されます。
- 距離減衰には対応していません。
