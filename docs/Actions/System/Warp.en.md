# Warp

![Warp](img/Warp.jpg)

The action "Warp" teleports the player to the designated object's position.

|  Label |  Function  |
| ----   | ---- |
| Target | Designates the warp target |

!!! Note Info
    - Rotation only applies to the Y-axis.
    - The avatar may fall due to gravity after warp.
