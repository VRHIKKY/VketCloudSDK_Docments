# Motion

![Motion](img/Motion.jpg)

Motionは[HEOWorldSetting](../../HEOComponents/HEOWorldSetting.md#_4)に設定されている[プリセットアバター](../../HEOComponents/HEOWorldSetting.md#_4)及び[マイアバター](../../HEOComponents/HEOWorldSetting.md#_5)のモーションを再生します。

`Motion Name`には、モーション名を指定します。

| 名称 | 機能 |
| ---- | ---- |
| Motion Name | アバターファイルに設定されたモーション名を設定してください |

Motionアクション発動時、プレイヤーが使用しているアバターに`Motion Name`で指定されたモーションがない場合、モーションは再生されません。

各アバターのモーション名とアニメーションはプリセットアバターの[Avatar File / Motion](../../WorldMakingGuide/AvatarFile.md#motion)もしくは[HEOWorldSetting / MyAvatar](../../HEOComponents/HEOWorldSetting.md#motion)から確認・変更できます。
