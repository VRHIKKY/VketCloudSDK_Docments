
# NextMotion
![NextMotion](img/NextMotion.jpg)

NextMotion lets you set a motion to play after the previous Motion Action is completed. You may not set multiple of these in one Action list.

|  Label |  Function  |
| ----   | ---- |
| Name | Set the motion's name. |
