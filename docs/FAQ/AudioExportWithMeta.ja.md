# 同封した.mp3音源ファイルがビルド時に出力されない

## 現象
アクティビティに同封した音源素材が、SDKビルド時のreleaseフォルダに入らないケースの対応方法。

## 原因

ID3という音源ファイルのメタ情報が、指定された規格に沿っていない。

SDKでチェックを通れるID3メタ情報は以下の通り。

| ID3バージョン | バイナリ　|
| --- | --- |
| ID3v1 | 54 41 47 |
| ID3v2.3 | 49 44 33 |
| ID3v2.4 | FF FB A0 |
| ID3v2.5 | FF FB B0 |

これを通過できない.mp3データが弾かれている。
